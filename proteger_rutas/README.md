# Food Store - Evaluación 1 (Programación III)

## 📝 Descripción
Este proyecto es una aplicación frontend para una tienda de comida que permite gestionar un catálogo y un carrito de compras dinámico.

## 🚀 Instrucciones para ejecución local
Para levantar el proyecto, ejecutá los siguientes comandos en la terminal:

1. **Instalar las dependencias:**
   ```bash
   pnpm install
   ```
2. **Iniciar el servidor de desarrollo:**
   ```bash
   pnpm dev
   ```
3. **Ver la aplicación:**
   Abrí en tu navegador: `http://localhost:5173`

## ✅ Funcionalidades implementadas
* Carrito con persistencia (LocalStorage).
* Búsqueda por nombre y filtrado por categoría.
* Cálculo del total de la compra.
